An all in one manager for Movies, Series, Anime and manga.

Uses these API's:  
[Trakt](https://github.com/trakt/api-help)

## Uses
* Kotlin
* MVI/MVVM Architecture
* Retrofit
* Coroutines
* Hilt
* Paging3
* MDC Library
* Room
* Jetpack Navigation
* WorkManager
* Coil
* [DotsIndicator](https://github.com/tommybuonomo/dotsindicator)
* LeakCanary
