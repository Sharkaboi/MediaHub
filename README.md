Uses the [MAL Api](https://myanimelist.net/apiconfig/references/api/v2)

## Uses
* Kotlin
* MVI/MVVM Architecture
* Retrofit
* Coroutines
* Hilt
* Paging3
* MDC Library
* Jetpack Navigation
* Coil
* LeakCanary
